# Victoria Travels Admin v2

Next.js 15 admin panel for Victoria Travels, rebuilt with modern architecture and premium UI.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Create a `.env.local` file:
```
NEXT_PUBLIC_API_BASE_URL=https://vtserver-okhz.onrender.com/api/
```

3. Run the development server:
```bash
npm run dev
```

The admin panel will be available at [http://localhost:9003](http://localhost:9003)

## Features

- Server-side pagination
- Server-side export (Excel/CSV)
- Premium UI with Radix UI components
- Server-side rendering
- Cookie-based authentication
- Admin role protection

## Tech Stack

- Next.js 15.3.3
- React 18
- TypeScript
- Tailwind CSS
- Radix UI
- Recharts
